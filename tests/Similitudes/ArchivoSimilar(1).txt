Para tener Python en tu computadora primero visita la pagina oficial https://www.python.org/downloads y baja la version que corresponda a tu sistema operativo En el caso de Windows abre el archivo descargado y marca la casilla "Add Python to PATH" antes de presionar "Install Now" Si usas macOS puedes instalarlo con el archivo descargado o mediante Homebrew usando el comando "brew install python" En sistemas Linux basados en Debian basta con ejecutar "sudo apt install python3" en la terminal Una vez terminado el proceso abre una consola y escribe "python --version" o "python3 --version" para asegurarte de que la instalacion fue exitosa
