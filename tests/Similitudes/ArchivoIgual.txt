Para instalar Python en tu computadora primero ve al sitio oficial https://www.python.org/downloads y descarga la version recomendada para tu sistema operativo En Windows abre el instalador y asegurate de marcar la opcion "Add Python to PATH" antes de hacer clic en "Install Now" En macOS puedes usar el instalador o instalarlo con Homebrew usando el comando "brew install python" En Linux normalmente puedes instalarlo desde la terminal con "sudo apt install python3" para distribuciones basadas en Debian Una vez instalado abre una terminal o consola y escribe "python --version" o "python3 --version" para verificar que la instalacion fue exitosa
